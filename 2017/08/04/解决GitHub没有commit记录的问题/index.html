解决GitHub没有commit记录的问题

2017.08.04 Reckless  热度 ℃

很早之前就注意这个问题了，怎奈工作比较忙，解决这个问题就被我搁置了，今天我咨询了一下大神如何解决这个问题。

不多BB，直接上解决问题思路

首先使用git命令git log打开提交历史

下面是我的提交的一条记录

  • commit bd33a14811b336cfbf474cbbcf7fa83aaa7b4d1a
    Author: lmz limuzi@8dol.com
    Date: Mon Apr 24 17:45:37 2017 +0800`

我们注意到 用户名是 lmz 邮箱是 limuzi@8dol.com 这个是我之前使用的用户名和git账号关联的邮箱

我们所要做的只要 将用户名改为GitHub的用户名，邮箱换为GitHub账号关联的邮箱即可~
